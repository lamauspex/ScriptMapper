# План разработки модулей для обработки сценариев

## Цель
Создать систему для обработки сценариев в форматах PDF и DOCX с последующим анализом содержания.

## Предлагаемая архитектура

### 1. Модуль загрузки сценария (script_loader.py)
**Функции:**
- Извлечение текста из PDF файлов (используя PyPDF2 или pdfplumber)
- Извлечение текста из DOCX файлов (используя python-docx)
- Структурирование текста в читаемый формат
- Сегментация на сцены по паттернам (например, "INT.", "EXT.", номера сцен)
- Маркировка каждой сцены

### 2. Модуль анализа (script_analyzer.py)
**Функции:**
- Извлечение персонажей (поиск имен в верхнем регистре)
- Определение локаций (ключевые слова INT./EXT.)
- Анализ реквизита (объекты и предметы в сцене)
- Определение стилистических особенностей (тон, жанр)
- Анализ настроения сцены

## Технические детали
- Язык: Python
- Основные библиотеки: PyPDF2, pdfplumber, python-docx, nltk/spacy для NLP
- Структура данных: JSON для вывода результатов
